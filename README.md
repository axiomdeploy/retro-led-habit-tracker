# Retro LED Habit Tracker

![HTML](https://img.shields.io/badge/Made%20with-HTML-%23e34f26.svg)

## Description

The **Retro LED Habit Tracker** is a web-based application designed to help you maintain habits. It provides retro-styled LED visuals to showcase your daily streaks and statistics, making habit tracking fun and visually engaging.

## Features

- **Streak Tracking**: Visualize your daily habit streaks.
- **Stats Dashboard**: Get insights into your progress with stats.
- **Retro Style**: Enjoy the nostalgia of a retro LED display.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/axiomdeploy/retro-led-habit-tracker.git
   cd retro-led-habit-tracker
   ```

2. Open any `.html` file in your preferred web browser:
   - Simply double-click the file or drag it to the browser.

3. Start tracking!
